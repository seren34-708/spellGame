# 🎮 VocaDash - RPG風英単語スペリングゲーム

モンスターを倒しながら英単語のスペリングを学ぶ、RPG風の学習ゲームです！

## 🌟 特徴

### ⚔️ RPGバトルシステム
- **モンスター討伐**: 10体のモンスターとバトル
- **レベル別モンスター**:
  - 🐸 レベル1（1-4問目）: カエル
  - 🐺 レベル2（5-7問目）: オオカミ
  - 🐉 レベル3（8-9問目）: ドラゴン
  - 👑 レベル4（10問目）: ラスボス

### 📚 3つの難易度
- **😊 かんたん**: 英検3級レベル（20秒/問）
- **😎 ふつう**: 英検2級レベル（20秒/問）
- **🔥 むずかしい**: 英検1級レベル（20秒/問）

### 🎯 ゲームシステム
- **進捗表示**: バトルシーンで討伐状況を視覚的に確認
- **ヒント機能**: 第一文字が最初から表示
- **スキップ機能**: 難しい問題は別の問題に変更可能（1回限り）
- **タイマー**: 残り10秒で表示、緊張感のある演出

### 🎵 RPG風BGM
- **メニューBGM**: ファンタジー風の冒険テーマ
- **バトルBGM**: 緊張感のある戦闘音楽
- **効果音**: レベルアップ、正解、不正解などの効果音完備

### 🎨 RPGファンタジーの世界観
- **草原フィールド**: 風に揺れる草、流れる雲
- **羊皮紙UI**: 中世ファンタジー風のデザイン
- **装飾**: ⚔️剣と🛡️盾の装飾

## 🎮 遊び方

### 基本ルール
1. 難易度を選択
2. 表示される日本語の意味から英単語を推測
3. アルファベットを入力してスペリング
4. 正解でモンスター討伐成功！
5. 10問全てクリアで冒険完了

### 操作方法
- **文字入力**: キーボードでアルファベット入力
- **回答**: Enterキーまたは「回答」ボタン
- **スキップ**: 「スキップ」ボタン（1回のみ）
- **ミュート**: 🔊ボタンでBGMオン/オフ
- **ホーム**: 🏠ボタンで難易度選択画面に戻る

### ヒント機能
- 第一文字が常に表示
- 単語の長さが文字ボックスの数で分かる
- ヒントテキストで追加情報

### タイマー
- 各問題20秒の制限時間
- 最初の10秒はタイマー非表示
- 残り10秒で赤く表示
- 残り10秒から意味の文字が拡大

## 🏆 ランクシステム

冒険完了後、成績に応じてランクが表示されます：

- **100%**: 👑 伝説の勇者！
- **80%以上**: ⚔️ 王国の守護者！
- **60%以上**: 🛡️ 熟練の戦士！
- **40%以上**: 🗡️ 見習い冒険者！
- **40%未満**: 💪 修行中！

## 📊 進捗表示

### バトルシーン
```
🏰 🐸 🐸 🐸 🐸 🐺 🐺 🐺 🐉 🐉 👑
```

- **🗡️**: 現在戦闘中
- **💀**: 討伐成功（正解）
- **🛡️**: 討伐成功のマーク
- **👹💔**: 討伐失敗（不正解/時間切れ）

### レベルアップ
5問目と8問目でレベルアップ通知が表示されます！

## 💻 技術仕様

### ファイル構成
- `index.html`: 単一HTMLファイル（CSS/JavaScript込み）
- 外部依存なし、ブラウザだけで動作

### 対応環境
- **PC**: Chrome, Firefox, Safari, Edge
- **モバイル**: iOS Safari, Android Chrome
- **必要要件**: JavaScript有効なモダンブラウザ

### 技術スタック
- **HTML5**: セマンティックマークアップ
- **CSS3**: アニメーション、グラデーション、Flexbox
- **JavaScript**: バニラJS（ライブラリ不要）
- **Web Audio API**: BGM・効果音生成

### PC
- 最大幅: 700px
- フル機能
- ⚔️🛡️装飾表示

### モバイル
- タッチ操作対応
- タイトル非表示（スペース確保）

## 🎵 音楽・効果音

### BGM
- **メニューBGM**: ファンタジー冒険テーマ（5秒ループ）
- **バトルBGM**: 緊張感ある戦闘音楽（1.9秒ループ）

### 効果音
- **正解音**: 上昇音階
- **不正解音**: 下降音階
- **時間切れ音**: 低音ブザー
- **レベルアップ音**: ファンファーレ
- **カウントダウン音**: ティック音

## 🔧 カスタマイズ

### 難易度調整
`timeLimit`変数で制限時間を変更可能（デフォルト: 20秒）

### 単語追加
`questionBank`オブジェクトに単語を追加可能：
```javascript
{
    answer: "example",
    meaning: "例",
    hint: "ex-"
}
```

### BGMカスタマイズ
`playMenuBGM()`と`playBGM()`関数でメロディを編集可能
